# Matt's 13/14

Simple trinket tracker for slots 13 and 14.

Tracks cooldowns, shows ready effects, and supports blacklist controls.

<img width="1153" height="486" alt="image" src="https://github.com/user-attachments/assets/3fc95c0b-5166-4d5f-b4e4-eb50946348b2" />


## Main Features

- Track trinket cooldowns (slots 13/14)
- Optional glow, bounce, and sound when ready
- Blacklist trinkets by name or item ID
- Built-in Edit Mode for moving icons
- Minimap button toggle
